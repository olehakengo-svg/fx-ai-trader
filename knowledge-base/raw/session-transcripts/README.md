# Session Transcripts — セッション記録の生データ

## 用途
Claudeセッションの重要な判断過程や議論の生記録を保存する。
wiki/sessions/ が要約済みセッションログであるのに対し、ここは詳細記録。

## 保存すべきもの
- 重大な設計判断の議論プロセス（賛否両論含む）
- バグ調査の詳細なデバッグログ
- 戦略パラメータ調整の試行錯誤記録
- ユーザーとの合意形成過程（GO/NO-GO判断の根拠）

## いつ保存するか
- セッション中に重大な判断が行われたとき
- 後で「なぜこの判断をしたか」を追跡したいとき
- 複雑な調査で中間結果を記録したいとき

## ファイル命名規則
`{date}-{topic}.md` 例: `2026-04-13-equity-reset-analysis.md`
