## 発見1: TVSMの致命的設計欠陥 — SL buffer不足

### 症状
REALISTIC/PESSIMISTIC では **100% SL、AvgHold 1秒**で全滅。
「エントリー瞬間にSLが発動する」即死パターン。

### 根本原因（算術）
| 項目 | OPT | REAL | PESS |
|---|---|---|---|
| Entry slip (片側) | 0.95p | 2.10p | 4.30p |
| 合成データATR(60s) ≈ | 1.0p | 1.0p | 1.0p |
| SL距離 = 1.2 × ATR | 1.2p | 1.2p | 1.2p |
| **SL buffer = SL距離 − Entry slip** | **+0.25p** | **−0.90p** | **−3.10p** |

REALISTIC以上で **SL buffer が負** になる。
つまり「エントリーした瞬間、mid価格がSLより下にいる」状態。
1bar の normal volatility (±0.4p) で確実にトリガー。

### 示唆
- `SL = k × ATR` の設計は slippage-aware ではない
- **正しい設計**: `SL = k × ATR + Entry_slip × 2` のような cost-aware buffer
- または「ATR が slip × 2 より大きい時のみ発射」gate

### 補足: OPTでの +36p も疑わしい
- 実現 R:R = 5.1 (設計 2.5 の 2倍)
- これは「小さい即死SL(-1.46p) × 多数 + 稀なTP(+7.45p) × 少数」の構造
- TP到達は **74%の試行で0bar SL** をサバイブした 26%に偶然訪れたもの
- サンプル外で WR が 20% に落ちれば即崩壊: EV = 0.2×7.45 − 0.8×1.46 = +0.32p (まだ正)、
  15% なら EV = 0.15×7.45 − 0.85×1.46 = −0.13p (負)

## 発見2: OFIMR は仮説段階で失敗

### 症状
全シナリオで PF < 0.5、WR 1.5%、99% SL ヒット。
合成データであっても **即死**。

### 考察
- 設計: OFI 2σ極端 + VWAP 1ATR乖離 → fade
- 実測: ほぼ全て SL 3秒以内ヒット
- 原因:
  1. **分布構築の先読み問題の逆**: OFI 極端検出は正しいが、極端の後「そのまま同方向継続」した。すなわち「MR仮説が合成データ上で成立せず、モメンタム継続が支配」
  2. **fade エントリーは逆張り**: 方向性相場では原理的に負け続ける
  3. SL = 極値 + 0.3×ATR は極値形成直後に再度極値更新されやすい

### 仮説の妥当性
- Chordia-Subrahmanyam (2004) は **日足** の研究
- Evans-Lyons (2002) は OFI → 価格の **順張り** 関係を示す
- **3分スケールでの fade** は学術的に支持薄。設計時の引用は過剰
- 合成データでも破綻 → 実データでも likely 破綻

### 判定
**OFIMR は廃棄候補。** 実データ検証のコストすら正当化しにくい。

## 発見3: VBP は判定不能

### 症状
全シナリオで N=0（エントリー条件が合成データで成立せず）。

### 考察
- 条件: 直近20秒以内の30分レンジブレイク + 押し目 + 反発3バー
- 合成データは5分周期のトレンド切替 + 2%確率spike → ブレイク発生頻度が不足
- **実データ検証必須** だが、判断材料ゼロ

### 判定
**保留。実tickデータ30日分で最低 N≥30 を確認してから再評価。**

---

## 横断的結論

### 現状評価
1. **TVSM**: 設計欠陥（SL buffer不足）修正必須。修正後も retail-scalping の根本エッジ有無は未検証
2. **VBP**: 実データ必要。優先度は TVSM 次点
3. **OFIMR**: 廃棄推奨。仮説が学術的に脆弱かつ合成で即破綻

### コストモデルの非線形性
| Scenario | TVSM 実収益 |
|---|---|
| OPT → REAL | +36.6p → −104.6p (**ΔPnL = −141p**) |

片道コストが 0.95p → 2.10p (×2.2) になるだけで、戦略が **黒字→全損** に反転。
これは典型的な「エッジ > コスト」の裏返しで、コスト見積もりの精度が生死を決める。

### Retail Micro-Scalp エッジの存在証明
**本BTは「エッジが存在する証拠」を提供できなかった。**
合成データで発火した TVSM/OFIMR は現実的コストで全損。VBP は未発火。
→ retail 領域でのマイクロスキャルプ・エッジは、まず **存在仮説を実データで検証** するべきで、実装を先行させる価値は薄い。

1. **TVSM の SL 設計修正**: cost-aware buffer (`SL = max(1.2×ATR, 2×entry_slip + 0.5×ATR)`)
   - さらに `ATR < 2×entry_slip` の時は発射しない gate 追加
2. **実OANDA tickデータ 30日分** 収集（現状の検証は全て合成データ、情報価値ほぼゼロ）

## v2 修正後の再検証 (2026-04-17 同日)

### TVSM に実装した2つの gate

```python
# strategies/micro_scalp/tvsm.py
entry_slip_price = self.cost.total_cost_pips * self.pip
sl_dist_atr = 1.2 * atr
sl_dist_cost_aware = 2.0 * entry_slip_price + 0.5 * atr
sl_dist = max(sl_dist_atr, sl_dist_cost_aware)

# Gate 1: ボラティリティ下限
if atr < 2.0 * entry_slip_price:
    return None

# Gate 2: R:R 下限
if tp_dist < sl_dist * 1.5:
    return None
```

### v2 結果 (TVSM)

| Scenario | 修正前 N | 修正前 PnL | 修正後 N | 修正後 PnL |
|---|---|---|---|---|
| OPT  | 42 | +36.6p | **0** | 0 |
| REAL | 43 | −104.6p | **0** | 0 |
| PESS | 43 | −186.3p | **0** | 0 |

### 解釈 — 「発火しない = 失敗」ではない

合成データの ATR(60s) ≈ 1.0p に対し、OPT でも `2×slip = 1.9p > ATR` のため gate で全拒否。
これは **正しい挙動**:

- 「エッジ < コスト」の市況では取引しないのが最適戦略
- retail 個人投資家が HFT 領域で「無理に取引する」のは期待値マイナス
- **戦略が自動で沈黙する** のは blow-up 回避の最重要安全弁

### v2 クオンツ評価

| 項目 | v1 | v2 |
|---|---|---|
| 即死SL率 (REAL) | 100% | 0% (発火しない) |
| ドローダウン無限大リスク | 高 | 事実上なし |
| リアルtick検証の準備 | 不可 | 可能 |
| 意思決定の数学的整合性 | 破綻 | 整合 |

### 実データ投入時の期待値
- 実FX tick の ATR(60s) は通常 2-5p (OANDA USD/JPY 実測)
- 東京時間早朝/NY午後の低ボラ帯は 1p 未満 → gate で自動停止
- ロンドン/NYオープン、経済指標前後は 3-8p → gate 通過でエントリー候補
- **結果として「高ボラ時のみ取引」= 流動性 + スプレッド有利時間帯に自動偏る**

## v3 高ボラ合成データでの逆検証 (2026-04-17)

低ボラ合成では全戦略が gate で拒否された。これが「壊れてるのでは？」ではなく
「正しく保守」であることを確認するため、ボラティリティ 3.75倍の合成データで再検証。

```python
# 高ボラ版: noise=gauss(0, 1.5×pip), drift 3倍, spike 5-12p幅
```

### 結果

| 戦略 | OPT | OPT 診断 | REAL |
|---|---|---|---|
| **TVSM** | **N=48, WR=41.7%, PF=1.48, +48.3p** | TP:SL=20:28, R:R=2.07, BE-WR=32.6% | N=0 (gate) |
| VBP    | N=0 | エントリー未発火 | N=0 |
| OFIMR  | N=172, WR=2.3%, PF=0.66, −185.3p | 98% SL, 仮説成立せず | N=0 (gate) |

### クオンツ解釈

1. **TVSM は正しく機能する**: 高ボラ条件で初めて positive edge (+48p)
   - 実現 WR 41.7% が break-even 32.6% を約9%上回る
   - 実FXの LDN/NY オープン、指標前後の ATR(60s)=3-5p 帯域に相当
   - **次段階**: OANDA 実tickで同時間帯のみ検証

2. **OFIMR は高ボラでも失敗**: WR 2.3% は break-even 3.5% すら下回る
   - 高ボラ下でも 98% SL ヒット → fade 仮説自体が合成データで反証
   - 合成データが真のOFI微細構造を持たない可能性はあるが、**学術根拠(Kyle/Chordia)も日足研究**
   - **判定: 廃棄候補**。3分 OFI fade 仮説に固執せず、別仮説を検討

3. **VBP は発火せず**: 合成データの構造と不適合
   - 30分レンジブレイク + pullback + 反発 という複合条件が厳しい
   - 実データ投入前にエントリー条件の緩和テストが必要

### v3 総括

| 戦略 | Tier | 推奨行動 |
|---|---|---|
| TVSM  | **B+ (Promising)** | 実tickデータ検証へ進む |
| VBP   | C (Unproven) | 条件緩和 → 再検証 |
| OFIMR | D (Likely Broken) | 仮説再構築または廃棄 |
