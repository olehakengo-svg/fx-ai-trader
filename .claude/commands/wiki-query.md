# Wiki Query — ナレッジベースに質問する

FXトレードシステムのナレッジベース（knowledge-base/wiki/）を横断検索し、引用付きで回答してください。

## 手順

1. **wiki/index.mdを読む** — 全体構造を把握

2. **関連するwiki/ページを検索** — 質問に関連する戦略ページ、コンセプトページ、意思決定ページを特定

3. **引用付きで回答** — 各主張の根拠として、どのwiki/ページのどの情報に基づいているかを明記
   例: 「bb_rsiのpost-cutoff WR=36.4% (出典: [[bb-rsi-reversion]])」

4. **回答をwiki/に保存** — 質問と回答をwiki/syntheses/に保存
   ファイル名: `query-YYYY-MM-DD-short-title.md`

5. **新しいリンクを追加** — 回答中に発見した新しい関連性があれば、該当ページに[[内部リンク]]を追加

## 質問の例
- 「bb_rsiの現在のリスクは何か？」
- 「摩擦を最も削減できるアクションは何か？」
- 「独立監査で覆された判断とその理由は？」
- 「vol_momentum_scalpが即死率0%の理由は？」
